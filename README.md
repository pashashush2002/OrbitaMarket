# OrbitaMarket

## Цель

Разработать проект для продажи заказов на различные виды обработки спутниковых снимков

## Стейкхолдеры

- Оператор ДЗЗ, аналитические компании и внутренние команды - покупка заказов, создание и пополнение счёта
- Администраторы - администрирование системы

## Роадмап

16-20 июня: анализ требований, составление данного документа
21-25: разработка PaymentsService
26-30: разработка OrdersService
1-5 июля: разработка связи между сервисами с помощью Kafka
6-9: написание APIGateway
10: написание тестов, проверка работоспособности программы
11: сканирование проекта на наличие уязвимостей
12: выкладывание в репозиторий проекта, прикрепление ссылки в поле ответа
Конец июля: защита проекта
Август-сентяябрь: дальнейшее развитие проекта, его поддержка
